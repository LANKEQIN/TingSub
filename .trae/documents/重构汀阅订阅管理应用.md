### 项目分析

这是一个基于 Expo React Native + Redux Toolkit + Tamagui 的订阅管理应用，主要功能包括：
- 订阅添加、编辑、删除
- 多币种支持（CNY/USD/JPY）
- 主题切换（亮色/暗色）
- 订阅统计与提醒

### 重构计划

#### 1. 代码结构优化
- **问题**：当前代码存在一些冗余和不一致的地方
- **解决方案**：
  - 统一文件命名规范（如 `payment_methods` → `paymentMethods`）
  - 优化 Redux slice 结构，减少重复代码
  - 提取通用工具函数到单独文件

#### 2. 性能优化
- **问题**：HomeScreen 组件过于庞大（超过 600 行）
- **解决方案**：
  - 将表单逻辑、日期处理等提取到自定义 hooks
  - 优化 useMemo 和 useEffect 依赖
  - 实现虚拟滚动处理大量订阅

#### 3. 可维护性提升
- **问题**：类型定义分散，部分类型重复
- **解决方案**：
  - 统一类型定义到 `types/` 目录
  - 完善 TypeScript 类型检查
  - 添加 JSDoc 注释

#### 4. 用户体验优化
- **问题**：部分交互不够流畅
- **解决方案**：
  - 优化动画过渡效果
  - 改进表单验证和错误提示
  - 增强无障碍支持

#### 5. 技术栈升级
- **问题**：部分依赖版本可能较旧
- **解决方案**：
  - 升级 Expo 到最新稳定版
  - 优化 Tamagui 配置
  - 考虑使用 Expo Router 替代 React Navigation

### 实施步骤

1. **第一阶段**：代码结构优化和类型统一
2. **第二阶段**：HomeScreen 组件拆分和性能优化
3. **第三阶段**：用户体验和交互优化
4. **第四阶段**：技术栈升级和测试

### 预期成果

- 更清晰的代码结构
- 更好的性能表现
- 更流畅的用户体验
- 更高的可维护性
